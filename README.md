# OpenAssets Studio

A modern web application built with TypeScript for managing and deploying AI-powered assets.

## Features

- 🚀 Fast and responsive interface
- 💾 Local development support
- 🔑 Easy API key configuration
- ⚡ Hot module reloading with Vite

## Quick Start

**Prerequisites:** Node.js (v16 or higher)

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Configure your API key by setting `GEMINI_API_KEY` in [.env.local](.env.local):
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`

## Build for Production

```bash
npm run build
```

## Project Structure

This is a TypeScript-based project (98.9% TypeScript) with a modern development setup.
